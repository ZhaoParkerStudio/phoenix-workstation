# SB+LVO新闻剪辑脚本写作工作流程

## 一、任务目标

根据外交部例行记者会等官方表态素材，写作符合电视新闻播出标准的 **SB+LVO 剪辑脚本**，供剪辑师直接执行。

最终产出物为一份完整的【口播+SB+LVO+字幕】脚本，时长通常为40-60秒。

## 二、核心概念

| 术语 | 含义 | 作用 |
|------|------|------|
| **SB**（Sound Bite） | 发言人原声片段 | 呈现核心表态，增强权威性和现场感 |
| **LVO**（Live Voice Over） | 配音旁白 | 补充背景、串联信息，配合画面播出 |
| **口播**（Standup/Lead-in） | 主持人导语 | 一句话交代新闻由头，引出SB |
| **去SB** | 插入SB的位置标记 | 提示剪辑师此处切入原声 |
| **OUTCUE** | SB结束时的最后几个词 | 提示剪辑师原声播到此处结束切出 |
| **CG**（Character Generator） | 字幕条 | 显示发言人姓名、职务及新闻标题 |
| **Heading** | 副标题（小字） | 次要信息或延伸信息 |
| **Content** | 主标题（大字） | 核心信息，当天最重要新闻点 |

## 三、写作步骤

### Step 1：仔细阅读素材
- 通读发言人完整表态内容
- 标注出最有新闻价值、最具传播力的语句
- 区分：哪些信息必须在SB中呈现，哪些可以放入口播或LVO

### Step 2：确定SB片段（约15-25秒）
- **选择标准**：立场最鲜明、语气最有力、逻辑最完整的段落
- **切入方式**：不必从发言人第一句开始，可从"我要強調的是""我們奉勸""中方鄭重聲明"等表态性语句切入
- **时长控制**：
  - 40秒成片 → SB约15-18秒
  - 60秒成片 → SB约20-25秒
  - 原则上不超过25秒，否则观众注意力下降

### Step 3：写作口播（约8-12秒）
- **只讲核心事实**：时间、发言人、针对事件、中方采取的措施
- **不要堆砌原则**：原则性表述留给SB去呈现
- **引出SB的OUTCUE**：口播最后应自然过渡到SB的关键词
- **去SB位置**：标注在口播结尾处，OUTCUE之前

### Step 4：确定OUTCUE（3-5个词）
- 选取SB段落**最后几个词**作为切出提示
- 优选情绪高点、警示性强的语句，如"必將付出代價""堅決反對""高度警惕"
- 非逻辑起点，而是能留下印象的收束词

### Step 5：写作LVO（约15-25秒）
- **补充SB里没有的信息**：追问、背景、关联事件、延伸立场
- **不重复SB内容**：可用"重申""再次強調"呼应，但避免整句重复
- **LVO结尾**：呼应OUTCUE的警示语，形成韵律回响
- 语速按每分钟240字计算，20秒约80字

### Step 6：标注剪辑素材
- 精准标注每个clip的时间码（开始-结束）
- SB clip：截取发言人原声段落的精确时间段
- LVO clip：列出可用的衬底画面素材（发布会现场、外景空镜、资料画面等）
- 时长总和应覆盖口播+SB+LVO的配比

### Step 7：写作CG字幕
```
<CG=橫題>
Heading:[副标题，延伸/次要信息]
Content:[主标题，核心信息]
Reporter:[姓名] 綜合報道
```
- **Content必须是当天最核心、最尖锐的新闻点**
- Heading作为补充，可放延伸事件或次重点

## 四、时长配比参考

| 成片时长 | 口播 | SB | LVO |
|---------|------|-----|-----|
| 40秒 | 8秒 | 15秒 | 17秒 |
| 60秒 | 10秒 | 20-22秒 | 28-30秒 |
| 75秒 | 12秒 | 25-30秒 | 33-38秒 |

## 五、标准模板

```
【口播】
[时间] [发言人职务+姓名]主持例行記者會。針對[事件]，[发言人]表示，[措施概述]。
(去SB)
(OUTCUE:"[SB最后一句话的结尾几个词]")

-LVO-
[补充信息：追问/背景/关联事件]。发言人重申/強調，[核心立場]。再次警告，[警示語]。

-END-

<Clip>[素材ID](开始时间-结束时间)</Clip>

```

## 六、必须避免的错误

| 错误 | 正确做法 |
|------|---------|
| 口播把所有信息塞满 | 口播只讲事实，原则留给SB |
| SB从头开始录（"有记者问…"） | 从"我要強調""我們奉勸"等切入 |
| LVO重复SB内容 | LVO讲新信息，用"重申"呼应即可 |
| OUTCUE选逻辑起点 | 选情绪高点，如"付出代价""高度警惕" |
| Heading/Content主副颠倒 | Content=主标题，Heading=副标题 |
| 总时长超时 | 按目标时长倒逼精简，40秒就说40秒的话 |
| clip时间码缺失或笼统 | 必须精确到秒，如(05:23:05-05:46:11) |

## 七、三不原则（一票否决）

1. **不重复**：口播、SB、LVO三者信息不重复（呼应除外）
2. **不超时**：按给定时长精准配比，秒秒必争
3. **不模糊**：每段素材精确到秒，不留"随便剪"的空间

## 八、范例对照

### ❌ 错误写法（信息重复、SB太长、LVO冗余）

**【口播】**
中國外交部發言人毛寧4日主持例行記者會。針對新西蘭個別議員不顧中方嚴重關切和堅決反對，執意竄訪台灣地區，毛寧表示，一個中國原則是國際社會普遍共識，中方根據相關法律決定對有關人員採取禁止入境等措施，並警告在台灣問題上踩線越界必將付出代價。

(去SB)
(OUTCUE:"…一個中國原則是國際社會普遍共識")

-LVO-
針對記者追問是否會禁止德國等其他國家竄訪台灣的議員入境，發言人重申，一個中國原則是國際社會普遍共識，也是國際關係基本準則，同樣是中德關係的政治基礎。中方希望德方恪守一個中國原則，不要藉台灣問題干涉中國內政。發言人再次強調，在台灣問題上踩踏紅線，一定會付出代價。

SB 毛寧（完整发言，约45秒）
(近期新西蘭個別議員不顧中方的嚴重關切和堅決反對，執意竄訪中國台灣地區，違反一個中國原則…(中间省略)…在台灣問題上踩線越界必將付出代價。)
```

**问题：口播已说原则，SB又说一遍；SB太长；OUTCUE选在中间而非结尾；LVO重复"一个中国原则是国际共识"。**
```

## 九、快速自查清单

- [ ] 口播是否只讲了事实，没有抢SB的台词？
- [ ] OUTCUE是否选在情绪高点，而非逻辑起点？
- [ ] SB是否从表态性语句切入（如"我要强调""我们奉劝"）？
- [ ] SB时长是否控制在15-25秒？
- [ ] LVO是否讲的是新信息，而非重复SB？
- [ ] LVO结尾是否呼应了OUTCUE？
- [ ] Heading/Content主副是否正确（Content=主标题）？
- [ ] 每个clip是否标注了精确时间码？
- [ ] 总时长是否符合要求？
- [ ] 是否违反了"三不原则"中的任何一条？
